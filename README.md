# Coven and Labyrinth of GALLERIA support

ガレリアの地下迷宮と魔女ノ旅団をプレイする時に楽できるようなものです。  
基本的にバイナリの配布は行いません。cloneして自前でビルドしてください。  


## GalleriaVita.exe

vitacheatでリーンフォースのアドレスを調べた後に、他の項目のアドレスを刑計算してくれるやつです。  
タブレットからgoogleリモートでPCを操作して使うために作りました。  
ほんとはWebアプリとかで作りたかった。  
  
## GalleriaVitaWeb

GalleriaVita.exeをJavaScriptでWebアプリで作りました。  
04WebServerでサーバー立ち上げて確認しました。

## FindFolder
Webアプリの検証を行うために作ったものです。  
C#によるCGIのサンプルです。  
全然ガレリアと関係ないですな。  
  

## Dependency
Visual studio 2019 C#
